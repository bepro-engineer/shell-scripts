# 関数コメントルール

## 目的

このファイルは、Shell スクリプトの関数コメントに関する規約を定義する。

## 関数コメントルール

- すべての関数には必ずコメントを付ける
- 既存関数の修正時も、新規関数の追加時も、関数コメントは省略しない
- コメント未記載の関数を新規追加しない
- 関数コメントは日本語で統一する
- コメント形式は必ず以下のテンプレートを使う

```sh
# ------------------------------------------------------------------
# 関数名　　：xxx
# 概要　　　：○○○
# 説明　　　：
#   ○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○
#   ○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○
#
# 引数　　　：説明
# 戻り値　　：説明
# 使用箇所　：説明
# ------------------------------------------------------------------
```
